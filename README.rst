How to play
-------------------------------------------------------------------------------
::

  vim '+source chaos-game.vim'


Keys:

* ``<space>`` - next step
* ``<CR>`` - next N step (default 100)
* ``+`` - increase N by 100
* ``-`` - decrease N by 100 (minimum 100)
